
# MENU CARD

It creates full stack application for ordering food.

## TECH USED

**Frontend:** JSP

**Backend:** Servlet

**ORM Framework:** Hibernate

**Database:** MySQL

## DEPENDENCIES

This project uses the following dependencies from maven repository,

Java Servlet API

Version: 3.1.0

Hibernate Core Relocation

Version: 5.4.32

MySQL Connector/J

Version: 8.0.16
